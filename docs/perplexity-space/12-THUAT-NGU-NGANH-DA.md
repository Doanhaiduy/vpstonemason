## 10. PHÂN BIỆT DỄ NHẦM

### Quartz ≠ Quartzite:
```
QUARTZ = Đá NHÂN TẠO (engineered), đồng nhất, non-porous
QUARTZITE = Đá TỰ NHIÊN (natural), mỗi tấm unique, cần seal
→ Rất hay bị nhầm! Cẩn thận khi tư vấn khách
```

### Honed ≠ Matte:
```
HONED = Đá tự nhiên được mài mịn, không đánh bóng, vẫn có depth
MATTE = Thuật ngữ chung cho bề mặt không bóng, thường dùng cho engineered/porcelain
→ Kết quả tương tự nhưng quy trình khác nhau
```

### Benchtop ≠ Countertop:
```
BENCHTOP = Tiếng Úc (Australian English) ← DÙNG TỪ NÀY
COUNTERTOP = Tiếng Mỹ (American English) ← tránh dùng cho khách Úc
→ Luôn dùng "benchtop" khi viết cho thị trường Úc
```

### Splashback ≠ Backsplash:
```
SPLASHBACK = Tiếng Úc ← DÙNG TỪ NÀY
BACKSPLASH = Tiếng Mỹ ← tránh dùng
```

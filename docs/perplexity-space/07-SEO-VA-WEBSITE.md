## 4. BLOG SEO CHECKLIST

Mỗi blog post phải có:
- [ ] **SEO Title**: 50-60 ký tự, chứa primary keyword
- [ ] **Meta Description**: 150-160 ký tự, có CTA
- [ ] **URL Slug**: Ngắn, chứa keyword
- [ ] **H1**: Duy nhất, chứa primary keyword
- [ ] **H2s**: 3-6 headings phụ, chứa secondary keywords
- [ ] **First 100 words**: Đề cập primary keyword
- [ ] **Image alt text**: Mô tả ảnh + keyword
- [ ] **Internal links**: 2-4 link đến trang khác
- [ ] **Word count**: 1,200-2,500 từ
- [ ] **Readability**: Đoạn ngắn, lists, tables

## 5. TECHNICAL SEO

- Page Speed < 3 giây (test: PageSpeed Insights)
- Mobile-Friendly 100% (test: Mobile-Friendly Test)
- Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1

### Setup cần thiết:
- **Sitemap**: `/sitemap.xml` ✅ (đã có)
- **Robots.txt**: `/robots.txt` ✅ (đã có)
- **Google Search Console**: Submit sitemap, monitor rankings
- **Google Analytics 4**: Track traffic, conversions
- **Google Business Profile**: Claim, optimize, post weekly

### Schema Markup cần thêm:
- LocalBusiness Schema (homepage)
- Product Schema (trang sản phẩm)
- FAQ Schema (blog posts)
- BreadcrumbList Schema (navigation)

## 6. LOCAL SEO

### Google Business Profile Setup:
1. Claim/Create listing tại business.google.com
2. Điền đầy đủ: Name, Address, Phone, Hours, Website
3. Categories: "Stone Supplier", "Kitchen Remodeler"
4. Thêm photos: showroom, projects, team
5. Post weekly: updates, offers
